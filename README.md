## User Registration

To register a new user, follow these steps:

1. Navigate to the registration page. (The specific URL will depend on your deployment.)
2. Fill in the required fields:
    - Email
    - Password
    - Name
    - Phone Number
3. Click the "Register" button.

Upon successful registration, you will be redirected to the login page.  Please note that:

* Email addresses must be unique.
* Passwords are stored securely using encryption.
* Real-time validation is performed on input fields.
